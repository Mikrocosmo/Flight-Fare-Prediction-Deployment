## Directory Structure
Your directory structure should look like this
```
-static
  -styles.css
-templates
  -flight_fare.html
  -result.html
-app.py
-Procfile
-Readme.md
-requirements.txt
-rf_flight.pkl
```

## Installation

 To install all requirements:
 
`pip install -r requirements.txt`
